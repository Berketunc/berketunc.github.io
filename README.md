# Personal Portfolio Website

This repository contains the source code for my personal portfolio website.

The site showcases my background in Computer Science and Mathematics, with a focus on
AI/ML, autonomous systems, and software engineering. It includes sections for projects,
skills, experience, and contact information.

## Tech Stack
- HTML5
- CSS3 (Custom design system)
- JavaScript (Vanilla)
- Responsive layout

## Features
- Modern, responsive design
- Custom color palette and typography
- Animated hero section
- Projects and skills dynamically rendered
- Mobile-friendly navigation
